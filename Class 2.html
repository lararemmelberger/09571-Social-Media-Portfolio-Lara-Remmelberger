Class 2: Platform Architecture and Infrastructure


Definitions

1.Platform Architecture:

- high-level structure and design of a computing platform
- encompasses the arrangement and interaction of software, hardware, networks,
and other components that make up the foundation for applications and services - defines how different elements of the platform work together to provide functionality and support applications

2.Infrastructure

- the underlying foundation of a system or organization, including hardware,
software, networks, facilities, and other essential components
- IT infrastructure provides the necessary support for information processing and communication
- includes servers, storage, networks, data centers, and other resources required for the operation
of applications and services.


Client / Server architecture

It is a computing model that describes the relationship between two types of devices or programs: clients and servers.
This architecture is widely used in computer networks and distributed systems.

Client / Server infrastructure

The client-server architecture relies on a complex infrastructure to enable
communication between clients and servers. This infrastructure includes various
components and technologies that support the functioning of the architecture.

Related Buzzwords:

DNS (Domain Name System)

DNS is a part of Internet Infrastructure that translates human-readable domain names into
numerical IP addresses that can be understood by computers. Because of this we can search for websites
in a webbrowser/search for the domain name(e.g.: www.google.com), instead of having to somewow look up their IP adresses.

Hypertext Transfer Protocol (HTTP):>

HTTP is a set of rules or protocols that enables communication between your web browser (like Chrome, Firefox, or Safari)
and web servers. It's the language that allows your computer to send requests and recieve asked information.
You usually send a http request when you want to acces a website through the Internet.

TLS (Transport Layer Security)

TLS is a protocol that helps secure communication over a computer network,
ensuring the privacy, integrity, and authenticity of data exchanged between two systems.
It encrypts Information that is being transferred in order to prevent 3rd partys from being able to steal information along the way.
