Class 3-5: Frontend Development

Definitions

Frontend development, often simply referred to as "front-end," is a discipline in web and software development
that focuses on creating the user interface and user experience (UI/UX) of a website or application. The primary goal
of frontend development is to ensure that the application or website is visually appealing, responsive, and user-friendly.
Frontend developers work on the client side of web development, which means they deal with what users see and interact with
in their browsers or on their devices.

Good UI/UX

A good user interface (UI) refers to the visual and interactive aspects of a product, whether it's a website, application, or any other software.
A UI is considered good when it effectively presents information and functionality to the user in an aesthetically pleasing and user-friendly manner.

User experience encompasses the broader, holistic experience a user has with a product, including their emotional and practical interactions.
A good user experience focuses on making the entire interaction as positive and efficient as possible. It includes the UI but extends beyond it.

Key Technologies (web development)

Web pages are composed by:
- HTML (Hypertext Markup Language) for structuring content;
- CSS (Cascading Style Sheets) for styling and layout;
- JavaScript for adding interactivity

Related Languages:

HTML (HyperText Markup Language)

"Hypertext" refers to links that connect web pages to one another,
either within a single website or between websites.
HTML uses "markup" to annotate text, images, and other content for display in a Web browser.
It is basically the fundamental language that Websites on the Internet are build with.

CSS (Cascading Style Sheets)

CSS is a style sheet language that helps selectively style html elements. As a simple example you can use CSS to color words.
CSS layouts refer to the way elements on a web page are structured and arranged, defining how they appear in terms of their position,
size, and relationship to one another.

Javascript

JavaScript is a versatile programming language primarily used for creating interactive and dynamic
content on websites. It runs in web browsers and allows developers to manipulate the Document Object Model (DOM),
respond to user actions, and enhance the overall user experience. It is used for web pages that do more than just sit
there and display static information for you to look at — displaying timely content updates, interactive maps,
animated 2D/3D graphics, scrolling video jukeboxes, etc.
