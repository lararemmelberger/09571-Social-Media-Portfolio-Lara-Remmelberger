Instagram: Request Types and Domain/Endpoints

Request Types

POST

- upload media, add comments
- Input: User-Id, Media content
- Output: Information about the uploaded media

PUT

- edit caption on posts
- edit comments
- Input: Media-Id
- Output: Information about the added comment

DELETE

- delete comments on posts
- delete posts
- delete direct messages
- delete profile/account
- Input: Media-Id, Comment-Id
- Output: Confirmation of the deleted comment

GET

- user profile, retrieve media, search media by hashtags, get comments on a post,
get comments on a post, get followers, retrieve story (etc.) - Input: User-Id - Output: Username, profile picture, follower/following count
