Class 10: Data Storage and Databases

Data Storage

Data storage refers to the physical or digital means of preserving and retaining data for future use.
It is the broader concept that encompasses various methods and technologies for storing data.
Data storage can include a wide range of storage technologies, such as hard disk drives (HDDs),
solid-state drives (SSDs), cloud storage, tape storage, optical storage, and more.

Databases

It is a structured collection of data organized for efficient storage, retrieval, and management.
It is designed to support the efficient querying and processing of data, enforcing relationships between different pieces of information.

Types of database (examples)

Relational Databases (RDBMS):

- Organize data into tables with rows and columns.
- Enforce relationships between tables.
- Examples: MySQL, PostgreSQL, Oracle Database, Microsoft SQL Server.

NoSQL Databases:

- Designed for flexibility and scalability.
- Do not rely on a fixed schema.

In-Memory Databases:

Store data in the system's main memory (RAM) for faster access.

Distributed Databases:

Distribute data across multiple nodes or servers for improved scalability and fault tolerance.

Related Buzzwords: ACID

ACID stands for:
1.Atomicity: Views the entire Data transaction as a single unit.
If theres an error with even just one element of that transaction, the entire transaction will be rolled back and the database will stay unchanged.
2. Consistency: Consistency ensures that a transaction brings the database from one valid state to another.
If a transaction violates any integrity constraints, the entire transaction is rolled back, and the database is restored to its previous state. 3.Isolation: Isolation ensures that the execution of one transaction is isolated from the execution of other transactions.
It prevents interference between transactions, which can lead to inconsistent or incorrect results. 4. Durability: Durability guarantees that once a transaction is committed, its effects will persist even in the event of a system failure. They are permanent.
